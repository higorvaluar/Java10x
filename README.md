
# Java10x Course Repository

Welcome to my repository dedicated to the Java course, which spans from beginner to advanced levels. Here you will find all the code, challenges, and notes related to the lessons, organized by difficulty level. Currently, I am at the Intermediate Level.

This course was created by **Fiasco** from the YouTube channel <a href="https://youtube.com/@GrandeFiasco" target="_blank">Grande Fiasco</a>, a channel with over 200,000 followers.

## Course Levels

### Beginner Level
Basic Java concepts explained in a simple and practical way. The fundamentals are covered with everyday examples to make learning easier.

- Variables and Data Types
- Conditional Structures and Loops
- Arrays and Garbage Collector
- Practical challenges, such as the "Ninja Registration"

### Intermediate Level
A deeper dive into essential concepts of object-oriented programming, including inheritance, polymorphism, encapsulation, and advanced data manipulation.

- Classes and Objects
- Inheritance, Interfaces, and Abstract Classes
- Data Structures (List, Stack, Queue, LinkedList, HashSet)
- Generics and Code Refactoring
- Practical challenges, such as "Bank in Konoha" and "Linked List of Ninjas"

### Advanced Level (In Progress)
Advanced topics to master Java with a more technical focus on performance and best practices.

- Algorithm Complexity and Big-O
- Static Scope and PSVM
- Code Performance Analysis

> **Note**: This level is still under development.
